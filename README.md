# Robocopy
A Windows Batch script using Robocopy to ensure reliable file transfers with progress tracking and error reduction, with DRAG and DROP.
## 🚀 How to Use
1. **Run the script:** Double-click `Robocopy-tool.bat`.
2. **Select Source:** Simply **drag and drop** your file directly into the command window and hit Enter.
3. **Select Destination:** **Drag and drop** the folder where you want the file to go and hit Enter.
4. **Done:** The script cleans up any extra quotes and starts the copy process with a progress bar!
